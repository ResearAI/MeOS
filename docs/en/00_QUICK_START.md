# 00 Quick Start

## What MeOS is

MeOS is a local-first personal operating-layer skill repository.

It helps you turn long-term working patterns into:

- reusable assets
- editable skill structure
- maintainable local knowledge

## What to do first

1. Read [`SKILL.md`](../../SKILL.md)
2. Read [`references/source-locations.md`](../../references/source-locations.md)
3. Read [`references/extraction-sop.md`](../../references/extraction-sop.md)
4. Read [`references/privacy-policy.md`](../../references/privacy-policy.md)

## Where to put things

- public method and templates:
  - `references/`
  - `schemas/`
  - `assets/templates/`
  - `assets/examples/`
- private local material:
  - `private/`
  - `evidence/`
  - `runtime/`
  - `assets/live/`

## First recommended workflow

1. keep your raw material local
2. inspect the templates
3. inspect the examples
4. create or update `assets/live/` only after promotion rules are satisfied

## What not to do

- do not commit secrets or private raw transcripts
- do not promote one-off behavior directly into a stable asset
- do not confuse evidence with reusable assets
