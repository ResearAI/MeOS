# 00 快速开始

## MeOS 是什么

MeOS 是一个本地优先的个人工作资产 skill 仓库。

它的目标不是“记住一切”，而是把长期稳定、可复用的工作方式整理成：

- 可读
- 可改
- 可维护

的 skill 资产。

## 第一步该做什么

1. 阅读 [`SKILL.md`](../../SKILL.md)
2. 阅读 [`references/source-locations.md`](../../references/source-locations.md)
3. 阅读 [`references/extraction-sop.md`](../../references/extraction-sop.md)
4. 阅读 [`references/privacy-policy.md`](../../references/privacy-policy.md)

## 目录怎么用

- 公开方法层：
  - `references/`
  - `schemas/`
  - `assets/templates/`
  - `assets/examples/`
- 本地私有层：
  - `private/`
  - `evidence/`
  - `runtime/`
  - `assets/live/`

## 推荐的第一次使用流程

1. 原始材料先放本地
2. 先看模板
3. 再看示例
4. 满足晋升条件后，再写入 `assets/live/`

## 不该做什么

- 不要把 secret 或私有原始对话直接提交进 git
- 不要把一次性行为直接升级成长期规则
- 不要把 evidence 和正式资产混在一起
